   - `fuscan cache stats`：显示缓存统计
   - `fuscan cache clear`：清空缓存（删除 DB 文件）
   - `fuscan cache prune [--max-age-days N]`：清理过期文件缓存

3. **`_cmd_scan` 变更**：
   ```python
   use_cache = config.cache_enabled and not args.no_cache
   if use_cache:
       cache_path = args.cache_path or default_cache_path()
       with CacheStore(cache_path) as cache:
           source_files = _compute_source_files(args.rules, not args.no_builtin)
           scanner = Scanner(ruleset, cache=cache, source_files=source_files, ...)
           report = scanner.scan(scan_path)
   else:
       scanner = Scanner(ruleset, ...)
       report = scanner.scan(scan_path)
   ```

4. **新增 `_compute_source_files`**：
   ```python
   def _compute_source_files(rules_paths: list[Path] | None, use_builtin: bool) -> dict[Path, str]:
       from fuscan.cache import hash_bytes
       sources: dict[Path, str] = {}
       if use_builtin:
           from fuscan.builtin import BUILTIN_RULES_PATH
           if BUILTIN_RULES_PATH.exists():
               sources[BUILTIN_RULES_PATH] = hash_bytes(BUILTIN_RULES_PATH.read_bytes())
       for p in rules_paths or []:
           if p.exists():
               sources[p] = hash_bytes(p.read_bytes())
       return sources
   ```

5. **新增 `_cmd_cache`**：处理 `cache` 子命令。

6. **`_cmd_tray` 变更**：构造 `CacheStore` 传给 `TrayApp`。

## 假设与决策

1. **覆盖率门槛 96%**：`pyproject.toml` `fail_under=96`，不得降低。
2. **IncrementalScanner 不传 source_files**：传 `None`，Scanner 内部用 `__inline__` 虚拟来源。规则哈希基于规则内容计算，不依赖文件路径。
3. **CacheStore 生命周期**：CLI 用 `with` 上下文管理；GUI 在 MainWindow 持有，closeEvent 关闭；Tray 在 TrayApp 持有。
4. **Python 3.8 兼容**：`from __future__ import annotations`；`CacheStore` 导入放 `TYPE_CHECKING` 块。
5. **state_file 参数保留**：TrayApp/CLI 的 `--state` 参数保留签名（空操作），避免破坏现有调用方。
6. **BUILTIN_RULES_PATH**：从 `fuscan.builtin` 导入内置规则文件路径计算 source_files。
